# 传票助手

> 自动从法院传票 PDF 中提取关键信息，并生成日历提醒。

适用于需要整理大量传票的法律工作者。**本工具完全本地运行，不上传任何数据。**

---

## 功能

- 自动识别传票中的**案号、案由、被传唤人、应到时间、应到处所**
- 识别结果由**真人逐项确认**后再处理，避免误操作【不想逐项确认也可改】
- 生成 `.ics` 日历文件，导入手机后自动设置**三档提醒**【想改的话直接py文件里改就行】
  - 开庭前 2 小时
  - 开庭前 1 天
  - 开庭前 1 周
- 生成 **HTML 索引页**，可直接点击打开对应传票 PDF
- 生成 **TXT 汇总**，便于存档

---

## 运行环境

- Python 3.10 及以上
- Windows / macOS / Linux

---

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/jmyi-jas/summons-extractor.git
cd summons-extractor
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行

```bash
直接运行，程序会提示输入路径
python 传票助手.py

# 方式二：如果你一定坚持的话……
python 传票助手.py "C:\我的传票文件夹"
```

### 4. 导入日历

将生成的 `传票日历.ics` 文件发送到手机，点击即可导入系统日历。
outlook等软件电脑上可以直接拖入，ics文件应用很广，相对方便

## 📁 文件命名规范

程序会自动扫描目标文件夹中**文件名含"传票"的 PDF**，例如：

```
✅ 传票_某某公司.pdf
✅ 2026-05-20传票.pdf
❌ 通知.pdf          ← 不含"传票"，跳过
```

一般来说法院的文件都会自动含传票，如果你还需要自动下载的程序，可以移步我的主页，有另一个快速下载的插件

## ⚠️ 兼容性说明

各地法院传票格式存在差异，程序采用「表格提取优先 + 正则兜底」策略。
如遇识别失败，可在确认环节手动填写，欢迎提 Issue 附上脱敏后的传票样本。
